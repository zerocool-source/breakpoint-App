# Breakpoint BI - Complete System Architecture

## Overview

Breakpoint BI is a business intelligence and automation platform for commercial pool management companies. It integrates with Pool Brain API to provide job management, chemical ordering automation, property communication channels, repair tracking, and commission calculations.

## API Endpoints

### Jobs & Alerts
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/jobs` | GET | Fetch all jobs with customer enrichment |
| `/api/jobs/:id/repairs` | GET | Parse office notes for parts/labor |
| `/api/alerts/enriched` | GET | Alerts with customer/pool context |
| `/api/alerts/chemical-order-email` | GET | Generate bulk chemical order |
| `/api/alerts/completed` | GET | Get completed alert IDs |
| `/api/alerts/completed` | POST | Mark alert as completed |

### Channels
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/channels` | GET | List all property channels |
| `/api/channels/sync` | POST | Sync channels from Pool Brain |
| `/api/channels/:id/messages` | GET | Get channel messages |
| `/api/channels/:id/messages` | POST | Send new message |
| `/api/channels/:id/messages/:msgId/replies` | GET | Get thread replies |
| `/api/channels/:id/messages/:msgId/reactions` | POST | Add reaction |

### Settings & Payroll
| Endpoint | Method | Description |
|----------|--------|-------------|
| `/api/settings` | GET/PUT | Pool Brain API credentials |
| `/api/payroll/periods` | GET | List pay periods |
| `/api/payroll/entries` | POST | Add job to payroll |
