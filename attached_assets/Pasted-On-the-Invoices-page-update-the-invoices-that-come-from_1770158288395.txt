**UPDATE INVOICES FROM WORK ORDERS:**

1. **ADD "WORK ORDER" TAG:**
- For invoices created from Work Orders, display a "Work Order" tag
- Tag color: Teal (#14b8a6) or orange (#f97316) to distinguish from other sources
- Position: In the CUSTOMER/WORK column, below the description (similar to how EST# is shown)

2. **ADD "SEND INVOICE" BUTTON:**
- For invoices with status "Draft" that came from Work Orders
- Show a "Send Invoice" button in the ACTIONS column
- Button style: Blue outline or solid button
- When clicked:

  - Get back the QB # (e.g., 1054, 1055)
  - Update status from "Draft" to "Sent"

**SEND INVOICE FUNCTIONALITY:**
```javascript
const sendInvoiceToQuickBooks = async (invoiceId) => {
  try {
    // 1. Send to QuickBooks API
    const response = await fetch('/api/invoices/send-to-quickbooks', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ invoiceId })
    });
    
    const result = await response.json();
    
    // 2. Update the invoice with QB data
    // - QB number (e.g., 1055)
    // - Status: "Sent"
    // - Sent date: today
    // - Emailed: customer email
    
    // 3. Show success toast
    toast.success(`Invoice sent to QuickBooks - QB #${result.qbNumber}`);
    
    // 4. Refresh the invoices list
    refreshInvoices();
    
  } catch (error) {
    toast.error('Failed to send invoice to QuickBooks');
  }
};
```
