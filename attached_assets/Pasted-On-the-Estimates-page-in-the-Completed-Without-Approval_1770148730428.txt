**WHEN CLICKING "CONVERT TO ESTIMATE":**

Pre-fill the Estimate form with:

1. **Customer/Property:** Auto-select the property
2. **Quote Details:**
   - Job Title: Work order description title
   - Quote Description: Full description text
3. **Line Items:** Pre-populate with parts used
   - Each part as a line item with description and amount
4. **Photos Section:**
   - Show "Before" photos with label
   - Show "After" photos with label
   - Photos should display as thumbnails
5. **Work Order Tracking:**
   - WO Number: Pre-filled
   - Work Order Received: Checked
6. **Repair Tech:** Pre-select the technician
7. **Estimate Total:** Sum of all line items
